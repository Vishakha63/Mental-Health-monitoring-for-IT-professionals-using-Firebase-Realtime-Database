# Mental Health Monitoring for IT Professionals using Firebase Realtime Database

## Overview

With the rising work pressure in the IT industry, the mental health of **71% of tech employees** indicates that mental health struggles hinder their productivity. This project provides a solution using **Firebaseâ€™s Realtime Database** to monitor and support the mental health of IT professionals in real-time.

The system includes **two web applications** built with **HTML, CSS, and JavaScript SDK**, both connected to the Firebase Realtime Database.

---

## Applications

### Application 1: Clinicians (Nurses or Health Workers)
- Access to **Add, Modify, Update, and Delete** employee data  
- Tracks **current mental health status, stress level, sleep hours, physical activity hours**, and other relevant metrics  

### Application 2: Doctors
- Provides **real-time data** of employees  
- Flexibility to **filter records** based on specific conditions  
- Enables doctors to monitor mental health trends and take necessary actions  

---

## Features

- **Real-time monitoring** of mental health data  
- **CRUD operations** for clinicians to manage employee records  
- **Data filtering and visualization** for doctors  
- Built using **Firebase Realtime Database** for instant updates  
- User-friendly **web interfaces** for both clinicians and doctors  

---

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript SDK  
- **Backend:** Firebase Realtime Database  
- **Data Handling:** Real-time storage and retrieval of employee mental health metrics  

---

## Usage

1. **Setup Firebase Project**: Configure Firebase and enable Realtime Database  
2. **Web Applications**:  
   - Clinicians can manage employee records  
   - Doctors can view and filter real-time data  
3. **Monitor Mental Health**: Collect and track employee stress, sleep, and activity data to provide timely support  

---

## Impact

This project aims to **enhance employee well-being**, reduce mental health-related productivity loss, and provide **actionable insights** to health professionals in IT organizations.
